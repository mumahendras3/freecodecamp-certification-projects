C Programming Language

One of the most widely used programming languages

Timeline of Language Development

  • 1972: Dennis Ritchie created C as an augmented version of Ken Thompson's B.
  • 1972 – 1978: C and Unix co-evolved with a goal of increasingly less assembly language in Unix and more portability.
  • 1978: Brian Kernighan and Dennis Ritchie published the first edition of The C Programming Language, known to C programmers as K&R, which describes a version of C commonly referred to as K&R C.
  • 1989: The American National Standards Institute (ANSI) ratified ANSI X3.159-1989 “Programming Language C”, standardizing the C language. This version of C is often referred to as ANSI C, Standard C, or sometimes C89.
  • 1990: The ANSI C standard (with formatting changes) was adopted by the International Organization for Standardization (ISO) as ISO/IEC 9899:1990, which is sometimes called C90.
  • 1999: The ISO/IEC 9899:1999 standard was published, revising the C standard further, which is commonly referred to as C99.
  • 2011: The ISO/IEC 9899:2011 standard was published, revising the C standard even further, which is commonly referred to as C11. The C standards committee adopted guidelines to limit the adoption of new features that had not been tested by existing implementations.
  • 2018: The ISO/IEC 9899:2018 standard was published, the current standard for the C programming language, which is commonly referred to as C17. It introduces no new language features, only technical corrections, and clarifications to defects in C11.

For more detail on the history of C, its features, syntax, characteristics, etc., see its Wikipedia entry.
